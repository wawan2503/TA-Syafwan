Kenapa pakai kalkulator ini?

Workflow singkat

Landing page ini dirancang supaya dosen/teman langsung paham apa yang tersedia.

  • Visual patch selalu tampil sehingga dimensi mudah dijelaskan.
  • Nilai return loss & VSWR sinkron otomatis dari Google Sheets.
  • UI bernuansa gelap biar nyaman saat presentasi malam hari.

Cara pakai

3 langkah sederhana

1

Pilih frekuensi

Pilih salah satu opsi preset dan lanjutkan dari tombol di atas.

2

Hitung otomatis

Masuk ke kalkulator, klik Hitung, dan dapatkan setiap parameter dimensi.

3

Tampilkan grafik

Scroll ke bagian grafik untuk menunjukkan performa return loss & VSWR.
